How to use
  • Install OBS websocket
  • Install StreamFX
  • Add a "Color Grading" filter to the source that you want to use
  • Fill in the details on the right and connect
  • Use any program that lets you route your audio to an output so that the browser can use it as a mic
  • Adjust the sliders to make the colors change according to the bars right above the silder's position example

Notes
  • Browsers won't allow the site to access the mic before you interact with the page. Refresh and click on the page right after it loads to fix it.
  • Sliders are not aligned perfectly with the bars.
  • Smoothing can't be adjusted in real time, refresh the site and reconnect to change it
  • Filter will not change if OBS is in Studio Mode
  • You will need OBS websocket >5.0.0 or OBS >28.0.0
