There is a list of geometric shapes. You will have to finish this list by adding more shapes.

The list is built according to a predefined hidden rule (e.g. "next shape cannot be the same color as the last one")

Use first shapes to figure out which rule is in order, then click on the shape which you think will fit.

This game does not have separate streamer and chat turns, you can play it together at the same time
